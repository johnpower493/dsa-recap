# Data Contracts (Senior-Level Focus)

This module focuses on **data contracts, schema governance, and product ownership**.

## Core Topics

- Schema versioning and breaking change strategy
- Consumer-driven contracts
- Data product ownership and stewardship
- SLAs and expectations

## Exercises

1. **Schema Versioning**
   - Create v1 and v2 schemas with a breaking change
   - Define migration strategy

2. **Contract Definition**
   - Write a contract for a “customer” dataset (schema + SLA)

3. **Ownership & Stewardship**
   - Define owners, consumers, and runbook steps
