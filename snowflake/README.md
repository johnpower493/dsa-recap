# Snowflake (Senior-Level Focus)

This module focuses on **platform engineering, cost governance, security, and performance tuning** in Snowflake.

## Core Topics

- Account + database architecture
- Roles, grants, RBAC/ABAC design
- Warehouse sizing + workload isolation
- Resource monitors and credit governance
- Query profiling + performance tuning
- Secure data sharing
- Clustering and pruning

## Exercises

1. **RBAC + Masking Policies**
   - Create roles and grant least privilege
   - Implement column masking for PII

2. **Warehouse Cost Controls**
   - Configure auto-suspend/resume
   - Set a resource monitor and test alerts

3. **Secure Data Sharing**
   - Create a share and grant it to a reader account

4. **Query Profiling**
   - Compare query performance with/without clustering
