## __Core Data Structures__

### 1. __Arrays & Strings__ (Most Fundamental)

- Two-pointer technique
- Sliding window
- Prefix sums
- Subarray manipulation

### 2. __Hash Maps / Hash Tables__ (Essential)

- Key-value lookups
- Counting frequencies
- Grouping/categorization
- Time complexity: O(1) average case

### 3. __Linked Lists__

- Singly linked list
- Doubly linked list
- Reversal
- Cycle detection (Floyd's algorithm)
- Fast/slow pointer technique

### 4. __Stacks & Queues__

- LIFO (Stack) vs FIFO (Queue)
- Monotonic stacks
- Priority queues (Heaps)
- Valid parentheses matching

### 5. __Trees__ (Critical)

- Binary trees
- Binary search trees (BST)
- Tree traversals: Inorder, Preorder, Postorder, Level-order (BFS)
- Tree properties: height, depth, size
- Common patterns: lowest common ancestor, path sum problems

### 6. __Graphs__ (Advanced but Essential)

- Adjacency matrix vs adjacency list
- DFS (Depth-First Search)
- BFS (Breadth-First Search)
- Shortest path: Dijkstra's algorithm
- Minimum spanning tree: Prim's & Kruskal's
- Topological sort
- Union-Find (Disjoint Set Union)

## __Core Algorithms__

### 1. __Sorting Algorithms__

- Merge Sort (O(n log n), stable)
- Quick Sort (O(n log n) average)
- Heap Sort
- Bubble, Insertion, Selection (for understanding basics)

### 2. __Searching Algorithms__

- Binary Search (you know this!)
- Linear Search (baseline)
- Two-dimensional search

### 3. __Dynamic Programming__ (Interview Favorite)

- Memoization (top-down)

- Tabulation (bottom-up)

- Common patterns:

  - Knapsack problem
  - Longest Common Subsequence
  - Longest Increasing Subsequence
  - Coin Change
  - Climbing Stairs/Fibonacci

### 4. __Greedy Algorithms__

- Activity selection
- Huffman coding
- Fractional knapsack
- Interval scheduling

### 5. __Two Pointers Technique__ (Highly Practical)

- Similar to binary search concept
- Used for: sorted arrays, palindrome checking, container problems

### 6. __Sliding Window__

- Fixed-size window
- Variable-size window
- Maximum/minimum in subarrays
- Substring problems

## __Special Patterns & Techniques__

### 1. __Recursion__

- Base cases
- Recursive case
- Tail recursion
- Backtracking (N-Queens, subsets, permutations)

### 2. __Bit Manipulation__

- XOR operations
- Bit shifting
- Counting set bits
- Power of two problems

### 3. __Divide and Conquer__

- Similar to binary search logic
- Split problem into smaller subproblems
- Merge results

### 4. __Math Algorithms__

- Greatest Common Divisor (GCD)
- Prime number sieve
- Modular arithmetic
- Fast exponentiation

## __Recommended Learning Order__

1. __Start with:__ Arrays, Strings, Hash Maps
2. __Then:__ Two pointers, Sliding window, Stacks/Queues
3. __Next:__ Linked Lists, Trees, Recursion
4. __Then:__ Binary Search (already done!), Sorting, Heaps
5. __Advanced:__ Graphs, Dynamic Programming, Greedy
6. __Mastery:__ Bit manipulation, Advanced math
